(SGM) Sistema de Gestion Marroqueria

El Sistema de Gestión Marroquería (SGM) es una herramienta integral diseñada para optimizar la administración de una marroquería. Este sistema permite gestionar de manera eficiente el inventario, facilitando el seguimiento y control de los productos disponibles. Además, el SGM ofrece funcionalidades para realizar ventas, registrando cada transacción de manera detallada. Una de las características clave del sistema es el historial completo de todas las actividades, lo que permite a los usuarios acceder a información histórica sobre ventas, entregas y movimientos de inventario. El SGM también incluye un sistema de notificaciones que avisa sobre entregas pendientes y pedidos atrasados, asegurando una gestión proactiva y un mejor servicio al cliente.

Sistema de actas para Arbolado

El Sistema de Actas para Arbolado es una solución digital diseñada para agilizar el proceso de llenado de formularios de poda en la sección de arbolado. Anteriormente, estos formularios se completaban en papel, lo que generaba demoras significativas en su gestión. Con este sistema, los inspectores pueden completar los formularios de manera rápida y eficiente. Una vez que el inspector llena el formulario, tiene la opción de confirmarlo y, de forma inmediata, se genera un archivo PDF que incluye todos los detalles necesarios: el formulario completo, la dirección del servicio realizado y la identificación del inspector responsable. Además, el sistema permite que el inspector firme el formulario digitalmente, asegurando la autenticidad del documento. Esta herramienta no solo ha reducido considerablemente el tiempo de gestión, sino que también ha mejorado la capacidad de respuesta hacia los vecinos, permitiendo a los inspectores atender un mayor número de solicitudes en menos tiempo. Con el Sistema de Actas para Arbolado, se ha optimizado el servicio, beneficiando a la comunidad de manera significativa.
